# 📖 Project Overview

This project demonstrates a complete **Enterprise-Level DevSecOps Implementation** for an Amazon Clone Ecommerce Inventory Management System built using **React**, **Django REST Framework**, and **SQLite**.

The primary objective of this project is to automate the complete Software Development Lifecycle (SDLC) while integrating security, containerization, orchestration, monitoring, and automated deployment.

The project follows DevSecOps best practices by integrating multiple security scanning tools into the CI/CD pipeline before deployment to Kubernetes.

✅ Build a Production-Ready Ecommerce Application
